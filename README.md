- Tên_kiểu là tên  của kiểu dữ liệu mà bạn muốn khai báo cho mảng

- Tên_biến là tên của mảng mà bạn muốn khai báo

- Spt: là số phần tử tối đa của mảng mà bạn muốn khai báo hay còn gọi là kích thức của mảng
